 📝 Student Attendance Manager – Java Console App

A simple and beginner-friendly Java project to manage student attendance using console input and a `HashMap`.

---

 🚀 Features

- Add multiple students at once
- Mark attendance for one or more students
- View all student attendance records
- View attendance percentage of any student
- Save all attendance to a text file on exit (`attendance.txt`)



 🛠️ Technologies Used

- Java (Core)
- HashMap, Array, ArrayList
- File Handling (`FileWriter`)
- Scanner for user input
- Console-based UI

---

 📂 How to Run

1. Clone the repo or download the `.java` file
2. Open it in any Java IDE (e.g., IntelliJ, VS Code)
3. Compile and run `StudentAttendanceManager.java`
4. Follow on-screen menu to add, mark, or view attendance

---

 📁 Output Example
--- Attendance records ---
Rahul - 2 days present
Priya - 3 days present
Vikram - 1 days present

On exit, it creates a file: attendance.txt
